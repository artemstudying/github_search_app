Приложение GitHub Search Tool выполнено с использованием технологий:
1. React.js
2. Js native стандарта ES6
3. css3, html5
4. Придерживался принципов Flat Design
5. GitHub
Содержит следующий функционал:
1. окно ввода для поиска реппозитория GitHub по ключевым словам
2. Возможность удаления елемента результатов поиска
3. При клике на выбранный репозиторий, его имя отображаеться в поле "You have selected repository".
4. При нажатии на кнопку "go to GitHub" в новом окне открывается страница выбранного репозитория на сайте github.com
